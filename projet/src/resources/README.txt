Pour que l'application fonctionne correctement, vous devez créer un dossier "resources" 
dans le même package que les fichiers Java, et y ajouter les images suivantes:

1. ciup_logo.png - Logo de la CIUP
2. default_house.jpg - Image par défaut pour les maisons
3. maison_internationale.jpg - Image pour la Maison Internationale
4. maison_japon.jpg - Image pour la Maison du Japon
5. maison_france.jpg - Image pour la Maison Française

Ces images peuvent être obtenues à partir du design Figma ou d'autres sources appropriées.
